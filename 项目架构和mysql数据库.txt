  常见问题排查

  本地环境无法访问

  # 1. 检查后端是否运行
  curl http://localhost:3020/health

  # 2. 检查端口占用
  netstat -ano | findstr :3020

  # 3. 查看错误日志
  type E:\123\sqlserver1-github\query-logs.txt

  线上环境无法访问

  # 1. 检查后端健康状态
  curl https://fgxny.onrender.com/health

  # 2. 检查MySQL外网连接
  timeout 5 bash -c 'cat < /dev/null > /dev/tcp/csfhcdz.f3322.net/3306' && echo "✓ MySQL可访问" || echo "✗
  MySQL不可访问"

  # 3. 查看Render日志
  # 访问 https://dashboard.render.com → Logs标签

  数据加载失败

  1. 检查Render日志：查看具体错误信息
  2. 检查环境变量：Render控制台 → Environment标签
  3. 检查MySQL外网访问：确保花生壳运行
  4. 检查MySQL用户权限：
  SELECT user, host FROM mysql.user WHERE user='external_user';
  SHOW GRANTS FOR 'external_user'@'%';
