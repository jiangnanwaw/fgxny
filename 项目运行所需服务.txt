  2. Node.js后端服务（3020端口）

  有多种启动方式，选择其中一种：

  # 双击运行

  方式2: 使用PM2（生产环境推荐）

  # 双击运行

  pm2 start ecosystem.config.js

  方式3: 使用npm命令

  cd E:\123\sqlserver1-github
  npm start

  方式4: 开发模式（带自动重启）

  验证服务状态

  注意事项

  1. MySQL必须先启动，否则后端服务会报错
  2. 端口3020必须未被占用
  3. SQL Server是远程数据库，需要网络连接
  4. 项目依赖已安装（如未安装，先运行 npm install）
